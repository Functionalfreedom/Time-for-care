## üîÑ Code Contribution Workflow

1. **Fork the Repository**
   - Use GitHub to fork the repository under your account.

2. **Clone Your Fork**
   ```bash
   git clone https://github.com/your-username/time-for-care.git
   cd time-for-care
   ```

3. **Create a New Branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```

4. **Make Your Changes**
   - Ensure your changes adhere to the project‚Äôs design principles.
   - Include meaningful commit messages.

5. **Run Tests (Optional in Early Phases)**
   - Ensure your code works as intended.

6. **Submit a Pull Request**
   - Push your branch to your forked repository.
   ```bash
   git push origin feature/your-feature-name
   ```
   - Open a pull request to the `main` branch of this repository.
